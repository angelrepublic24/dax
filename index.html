Microservicios

Son servicios pequeños que forman parte de un sistema mayor con una función específica.

Location Services:

Permite tener un sistema de localización de alto alcance ya que puede facilitar la búsqueda desde los continentes,países, estados de forma rápida. factible para agencias de viajes y ubicación.

Roster Services:

Permite crear las nóminas empresariales, y llevar un reporte de las mismas agregando los datos requeridos para que el sistema complete correctamente su función, siendo factible para cualquier empresa que desee llevar un control monetario de sus nominas

Monetization Services:

Este microservicio permite tener un desglose de las monedas y sus denominaciones de las mismas,Ej: DOP, USD. Siendo factible para los cierres de caja y otros.

Mailgum Services:

Este microservicio permite de forma rápida integrar tu mailgum al sistema al evitar la codificación de java en cuanto al envío de correos. comunicándose con tu cuenta de mailgum..
